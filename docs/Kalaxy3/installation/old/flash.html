#Ubuntu Installation and Node Provisioning

This document describes how to install Ubuntu on both Raspberry Pi arm64 nodes and amd64 bare-metal machines, apply cloud-init configuration, validate first boot, and prepare each node for cluster use.

For new installs, prefer Ubuntu 24.04 LTS unless a specific compatibility reason requires another release. Ubuntu provides Raspberry Pi preinstalled server images for modern Pi boards, and standard Ubuntu Server ISO images for amd64 machines. The Raspberry Pi preinstalled images include a cloud-init seed on the boot partition. For amd64 installs, cloud-init can be provided through autoinstall, NoCloud seed media, or post-install provisioning.
